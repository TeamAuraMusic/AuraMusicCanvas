AuraMusicCanvas

Public, no-auth Spotify Canvas video manifest for AuraMusic and compatible players.

Endpoint

GET https://canvas.auramusic.site/canvas.json

Format

{
  "version": 1,
  "updatedAt": "2026-05-24T...",
  "items": [
    {
      "song": "Title",
      "artist": "Artist",
      "url": "https://canvaz.scdn.co/...mp4"
    }
  ]
}

How to contribute

  1. Go to TeamAuraMusic/AuraMusicCanvas
  2. Use node scripts/add-canvas.js or edit public/canvas.json directly
  3. Open a Pull Request

Submissions are reviewed before being merged.

Submit via API (experimental)

POST JSON to /api/submit:

{
  "song": "Song Name",
  "artist": "Artist Name",
  "url": "https://...",
  "trackId": "optional-spotify-id"
}
